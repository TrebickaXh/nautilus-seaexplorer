

## Pause Task Generation: Archive All Routines and Clear Pending Tasks

### What will happen

1. **Archive all task routines** - Set `archived_at = now()` and `active = false` on all ~60+ routines. This immediately stops `materialize-tasks-v2` from creating new task instances (it only processes routines where `archived_at IS NULL`).

2. **Delete all 1,271 pending task instances** - Removes all future pending tasks to free up space.

### What will be kept

- Completed/skipped task instances (historical records)
- Completions (historical records)
- All configuration: shifts, locations, departments, users

### Technical Steps

**Data operations:**

```text
1. UPDATE task_routines SET archived_at = now(), active = false WHERE archived_at IS NULL
2. DELETE FROM task_instances WHERE status = 'pending'
```

### Result

- No new tasks will be generated by the materializer
- ~1,271 pending rows freed
- All routines preserved (just archived) -- you can reactivate them later when ready
